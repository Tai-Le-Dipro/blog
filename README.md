## Getting Started

#### Using with Direct Clone

- clone this repository
  ```bash
  git clone https://github.com/levinhtuyen/myblog
  ```
- install dependencies
  ```bash
  yarn install
  ```
- run development server
  ```bash
  yarn dev
  ```
- run build server
  ```bash
  yarn build
  ```
